سورة النبأ 

Сура «ан-Наба»  (Весть)

بِسْمِ اللّهِ الرَّحْمنِ الرَّحِيمِ

Во имя Аллаха Милостивого, Милосердного!

{ عَمَّ يَتَسَآءَلُونَ } * { عَنِ ٱلنَّبَإِ ٱلْعَظِيمِ } * { ٱلَّذِي هُمْ فِيهِ مُخْتَلِفُونَ } * { كَلاَّ سَيَعْلَمُونَ } * { ثُمَّ كَلاَّ سَيَعْلَمُونَ } * { أَلَمْ نَجْعَلِ ٱلأَرْضَ مِهَٰداً } * { وَٱلْجِبَالَ أَوْتَاداً } * { وَخَلَقْنَاكُمْ أَزْوَاجاً } * { وَجَعَلْنَا نَوْمَكُمْ سُبَاتاً } * { وَجَعَلْنَا ٱلَّيلَ لِبَاساً } * { وَجَعَلْنَا ٱلنَّهَارَ مَعَاشاً } * {وَبَنَيْنَا فَوْقَكُمْ سَبْعاً شِدَاداً } * { وَجَعَلْنَا سِرَاجاً وَهَّاجاً } * { وَأَنزَلْنَا مِنَ ٱلْمُعْصِرَاتِ مَآءً ثَجَّاجاً } * { لِّنُخْرِجَ بِهِ حَبّاً وَنَبَاتاً } * { وَجَنَّاتٍ أَلْفَافاً }
 1. О чем они расспрашивают друг друга?  2. О великой вести,  3. относительно которой они расходятся во мнениях.  4. Но нет, они узнают!  5. Еще раз нет, они узнают!  6. Разве Мы не сделали землю ложем,  7. а горы - колышками?  8. Мы сотворили вас парами,  9. и сделали ваш сон отдыхом,  10. и сделали ночь покрывалом,  11. и сделали день жалованием,  12. и воздвигли над вами семь твердынь,  13. и установили пылающий светильник,  14. и низвели из облаков обильно льющуюся воду,  15. чтобы взрастить ею зерна и растения  16. и густые сады. 
Аллах говорит о многобожниках, которые задают вопросы о Дне воскресения, отрицая его наступление:
{ عَمَّ يَتَسَآءَلُونَ عَنِ ٱلنَّبَإِ ٱلْعَظِيمِ } 
«О чем они расспрашивают друг друга? О великой вести». То есть: они расспрашивают друг друга о Воскресении, которое является великой вестью. Иначе говоря, они расспрашивают об ужасном, страшном и сокрушающем известии. Катада истолковал великую весть как воскрешение после смерти. По мнению Муджахида, это Коран. Но верным является первое толкование, так как в следующем аяте сказано: 
{ ٱلَّذِى هُمْ فِيهِ مُخْتَلِفُونَ }
«относительно которой они расходятся во мнениях». Это значит, что среди людей одни верят в это, а другие нет. Далее Аллах угрожает тем, которые отрицают День Воскресения: 
{ كَلاَّ سَيَعْلَمُونَ ثُمَّ كَلاَّ سَيَعْلَمُونَ }
«Но нет, они узнают! Еще раз нет, они узнают!». Это сильная угроза и явное предупреждение. 
Затем Всевышний Аллах говорит о том, что Он обладает великой силой, чтобы создавать удивительные вещи и изумительные события. Это доказывает Его могущество сделать все, что Он пожелает относительно Иного мира и других вопросов. Он сказал: 
{ أَلَمْ نَجْعَلِ ٱلأَرْضَ مِهَـٰداً }
«Разве Мы не сделали землю ложем», т.е. ровным, прочным и спокойным местом, подчинив ее вам. 
{ وَٱلْجِبَالَ أَوْتَاداً }
«А горы — колышками?», т.е. сделали их ее (земли) опорами, которые удерживают ее и не позволяют ей колебаться вместе с вами. 
Аллах говорит:
{ وَخَلَقْنَـٰكُمْ أَزْوَٰجاً }
«Мы сотворили вас парами», т.е. мужчину и женщину, чтобы вы радовали друг друга и воспроизводили потомство. Как еще сказал Аллах: 
{ وَمِنْ ءايَـٰتِهِ أَنْ خَلَقَ لَكُم مِّنْ أَنفُسِكُمْ أَزْوَٰجاً لِّتَسْكُنُوۤاْ إِلَيْهَا وَجَعَلَ بَيْنَكُم مَّوَدَّةً وَرَحْمَةً }
«Среди Его знамений — то, что Он сотворил из вас самих жен для вас, чтобы вы находили в них успокоение, и установил между вами любовь и милосердие» (30:21). 
Слова Аллаха:
{ وَجَعَلْنَا نَوْمَكُمْ سُبَاتاً }
«И сделали ваш сон отдыхом», т.е. временным прерыванием активной жизнедеятельности, чтобы вы могли отдохнуть от дневных хлопот. 
{ وَجَعَلْنَا ٱلَّيْلَ لِبَاساً }
«И сделали ночь покрывалом», т.е. ночь покрывает людей своим мраком и чернотой, как еще сказал Аллах: 
{ وَٱلَّيْلِ إِذَا يَغْشَـٰهَا }
«Клянусь ночью, которая скрывает его!» (91:4). Катада прокомментировал:
{ وَجَعَلْنَا ٱلَّيْلَ لِبَاساً }
 «"И сделали ночь покрывалом" значит "сделали ночь покоем"». 
Слова Аллаха:
{ وَجَعَلْنَا ٱلنَّهَارَ مَعَاشاً }
«И сделали день жалованием», т.е. сделали его сияющим, озаряющим, освещающим, чтобы люди в это время суток могли свободно перемещаться и заниматься своими рабочими делами. 
Слова Аллаха:
{ وَبَنَيْنَا فَوْقَكُمْ سَبْعاً شِدَاداً }
«И воздвигли над вами семь твердынь», т.е. семь пространных, высоких и совершенных небес, украшенных движущимися и неподвижными звездами. Поэтому Аллах сказал: 
{ وَجَعَلْنَا سِرَاجاً وَهَّاجاً }
«и установили пылающий светильник», имеется в виду излучающее на весь мир свет солнце, которое светит всем на Земле. 
Слова Аллаха:
{ وَأَنزَلْنَا مِنَ ٱلْمُعْصِرَاتِ مَآءً ثَجَّاجاً }
«И низвели из облаков обильно льющуюся воду». Ибн 'Аббас сказал, что
(المعصرات)
" аль-му'асыраат " (Тогда аят выглядит так: " И низвели посредством ветра обильно льющуюся воду" ) Ибн Абу Хатим передал от Ибн Аббаса: " И низвели посредством ветра". Такого же мнения придерживались Икрима, Муджахид, Катада и др. Утверждая, что это ветер, который заставляет дождь обильно изливаться из облаков. 'Али ибн Абу Тальха сказал, ссылаясь на ибн 'Аббаса:  «Это значит "низвели из облаков"». То же самое сказали 'Икрима, ад-Даххак, аль-Хасан, ар-Раби' ибн Анас и ас-Саури. Такой же комментарий выбрал ибн Джарир. Аль-Фараа сказал: «аль-му'асыраат» это облака, которые налились дождем, но еще не изливают дождь. Подобно тому, как девочку, у которой вот вот начнутся месячные, но она еще не менструирует, тоже называют «аль-му'асыраат». Катада же сказал : «Т.е с небес». Но это странные слова. Очевидно, что здесь имеются в виду облака. Подобно этому Аллах сказал: 
{ ٱللَّهُ ٱلَّذِى يُرْسِلُ ٱلرِّيَـٰحَ فَتُثِيرُ سَحَاباً فَيَبْسُطُهُ فِى ٱلسَّمَآءِ كَيْفَ يَشَآءُ وَيَجْعَلُهُ كِسَفاً فَتَرَى ٱلْوَدْقَ يَخْرُجُ مِنْ خِلاَلِهِ }
«Аллах — Тот, Кто посылает ветры, которые поднимают облака. Он простирает их по небу, как пожелает, и разрывает их в клочья. Потом ты видишь, как из их расщелин льется дождь» (30:48). Относительно слов Аллаха:
{ مَآءً ثَجَّاجاً }
«обильно льющуюся воду», Муджахид сказал: 
 (ﺛﺠﺎﺟﺎ)

«( саджаджа) изливающихся».  Ас-Саури истолковал: «Продолжающихся». Ибн Зейд пояснил: «Обильных». Я (Ибн Касир ) говорю:  Это слово "ﺍﻟﺜﺞ " встречается в хадисе о менструации где Зайнаб бинт Джахш сказала: «О посланник Аллаха, поистине, я женщина, у которой сильное хроническое кровотечение, что ты велишь мне делать? Это удерживает меня от молитвы и соблюдения поста». Он (да благословит его Аллах и приветствует), сказал:  «Я скажу тебе что поистине вата, удаляет ее». Она сказала: «Ее намного больше». Он сказал: «Тогда следует покрыть материалом половой орган». Она сказала: «Ее намного больше». Он сказал: «Тогда используй одежду». Она сказала: «Ее намного больше, она у меня 
( ﺃﺛﺞ ﺛﺠﺎ ) "
 поступает обильно" ». И это указывает на то, что
 (الثج)
 является тем, что много и обильно изливаться. Аллаху же об этом известно лучше!
{ لِّنُخْرِجَ بِهِ حَبّاً وَنَبَاتاً وَجَنَّـٰتٍ أَلْفَافاً }
«Чтобы взрастить ею зерна и растения и густые сады», т.е. чтобы взрастить этой обильной,  благой, полезной и благословенной водой 
{ حَبّاً }

«зерна», идущие в потребление людям и скоту. 
{ وَنَبَاتاً }
 «и растения», а именно: овощи, употребляемые в пищу в сыром виде,  
{ وَجَنَّـٰتٌ } 
 «и сады», где выращиваются разнообразные по цвету, запаху и вкусу фрукты, даже если они собраны в одной местности. Поэтому Аллах сказал: 
(وجنات ألفافاً)
«густые сады». Ибн 'Аббас и другие толкователи прокомментировали: «Это значит "собранные сады"». Подобно этому Аллах сказал:
{ وَفِى ٱلأَرْضِ قِطَعٌ مُّتَجَـٰوِرَٰتٌ وَجَنَّـٰتٌ مِّنْ أَعْنَـٰبٍ وَزَرْعٌ وَنَخِيلٌ صِنْوَٰنٌ وَغَيْرُ صِنْوَٰنٍ يُسْقَىٰ بِمَآءٍ وَٰحِدٍ وَنُفَضِّلُ بَعْضَهَا عَلَىٰ بَعْضٍ فِى ٱلأُكُلِ إِنَّ فِى ذٰلِكَ لآيَـٰتٍ لِّقَوْمٍ يَعْقِلُونَ }
 «На земле есть сопредельные участки, виноградные сады, посевы, пальмы, растущие из одного корня или из разных корней. Их орошают одной водой, но одни из них Мы создаем более вкусными, чем другие. Воистину, в этом — знамения для людей разумеющих» (13:4).
{ إِنَّ يَوْمَ ٱلْفَصْلِ كَانَ مِيقَاتاً } * { يَوْمَ يُنفَخُ فِي ٱلصُّورِ فَتَأْتُونَ أَفْوَاجاً } * { وَفُتِحَتِ ٱلسَّمَآءُ فَكَانَتْ أَبْوَاباً } * { وَسُيِّرَتِ ٱلْجِبَالُ فَكَانَتْ سَرَاباً } * { إِنَّ جَهَنَّمَ كَانَتْ مِرْصَاداً } * { لِّلطَّاغِينَ مَآباً } * { لاَّبِثِينَ فِيهَآ أَحْقَاباً } * { لاَّ يَذُوقُونَ فِيهَا بَرْداً وَلاَ شَرَاباً } * { إِلاَّ حَمِيماً وَغَسَّاقاً } * { جَزَآءً وِفَاقاً } * { إِنَّهُمْ كَانُواْ لاَ يَرْجُونَ حِسَاباً } * { وَكَذَّبُواْ بِآيَاتِنَا كِذَّاباً } * { وَكُلَّ شَيْءٍ أَحْصَيْنَاهُ كِتَاباً } * { فَذُوقُواْ فَلَن نَّزِيدَكُمْ إِلاَّ عَذَاباً }
17. Воистину, День различения назначен на определенное время.  18. В тот день затрубят в Рог, и вы придете толпами,  19. и небо раскроется и станет вратами,  20. и горы придут в движение и станут маревом.  21. Воистину, Геенна является засадой  22. и местом возвращения для тех, кто преступает границы дозволенного.  23. Они пробудут там долгие годы,  24. не вкушая ни прохлады, ни питья,  25. а только кипяток и гной.  26. Это будет подобающим возмездием.  27. Воистину, они не ожидали расчета  28. и полностью отрицали Наши знамения.  29. Мы же всякую вещь подсчитали и записали.  30. Вкушайте же! Мы не прибавим вам ничего, кроме мучений.
Всевышний Аллах сообщает, что День различения (День Воскресения) имеет свой установленный срок и он наступит ни секундой раньше, ни секундой позже. Никто не знает, когда придет этот Час, кроме Всемогущего и Великого Аллаха. Как еще сказал Аллах: 
{ وَمَا نُؤَخِّرُهُ إِلاَّ لاَِجَلٍ مَّعْدُودٍ } 
«Мы отсрочим его лишь до определенного срока» (11:104). 
{ يَوْمَ يُنفَخُ فِى ٱلصُّورِ فَتَأْتُونَ أَفْوَاجاً }
«В тот день затрубят в Рог, и вы придете толпами». Муджахид сказал: «Группа за группой». Ибн Джарир сказал: «Каждый народ придет со своим посланником». Подобно этому Аллах сказал: 
{ يَوْمَ نَدْعُواْ كُلَّ أُنَاسٍ بِإِمَـٰمِهِمْ }
«В тот день Мы созовем всех людей вместе с их предводителями» (17:71). Аль-Бухари сообщил со ссылкой на Абу Хурайру, что Посланник Аллаха (да благословит его Аллах и приветствует) сказал: 
" ما بين النفختين أربعون "
«Между двумя Трубными гласами сорок». Его спросили: «Сорок дней?» Он ответил: 
" أبيت "
«Я отказываюсь так утверждать». Его спросили: «Сорок месяцев?» Он ответил: 
" أبيت "
«Я отказываюсь так утверждать». Его спросили: «Сорок лет?» Он ответил: 

" أبيت "
«Я отказываюсь так утверждать». И добавил: 
 " ثم ينزل الله  من السماء ماء، فينبتون كما ينبت البقل، ليس من الإنسان شيء إلا يبلى إلا عظماً واحداً، وهو عجب الذنب، ومنه يركب الخلق يوم القيامة "
«Затем Аллах ниспошлет с неба воду, и они вырастут, как зелень. От человека после разложения не остается ничего, кроме одной кости — копчика, из которого люди будут воссозданы в День Воскресения» ( «Сахих». Фатх аль-Бари 8/558 )
{ وَفُتِحَتِ ٱلسَّمَآءُ فَكَانَتْ أَبْوَٰباً }
«И небо раскроется и станет вратами», т.е. дорожками, по которым спустятся ангелы.  
Слова Аллаха:
{ وَسُيِّرَتِ ٱلْجِبَالُ فَكَانَتْ سَرَاباً }
«И горы придут в движение и станут маревом», подобны словам Его: 
{ وَتَرَى ٱلْجِبَالَ تَحْسَبُهَا جَامِدَةً وَهِىَ تَمُرُّ مَرَّ ٱلسَّحَابِ }
«И ты увидишь, что горы, которые ты считал неподвижными, придут в движение, словно облака» (27:88), а также:
{ وَتَكُونُ ٱلْجِبَالُ كَٱلْعِهْنِ ٱلْمَنفُوشِ }
« а горы будут подобны расчесанной шерсти» (101: 5). И как сказано здесь.
{ فَكَانَتْ سَرَاباً }
«и станут маревом» - т.е. станут едва различимыми для смотрящего, словно мираж, а затем совсем исчезнут без следа. Подобно этому Аллах сказал: 
{ وَيَسْـئَلُونَكَ عَنِ ٱلْجِبَالِ فَقُلْ يَنسِفُهَا رَبِّى نَسْفاً فَيَذَرُهَا قَاعاً صَفْصَفاً لاَّ تَرَىٰ فِيهَا عِوَجاً وَلاۤ أَمْتاً } 
«Они спрашивают тебя о горах. Скажи: "Мой Господь развеет их и оставит только гладкую равнину, на которой ты не увидишь ни углубления, ни возвышения"» (20:105-107), а также: 
{ وَيَوْمَ نُسَيِّرُ ٱلْجِبَالَ وَتَرَى ٱلأَرْضَ بَارِزَةً }
«В тот день Мы заставим двигаться горы, и ты увидишь, что земля станет плоской» (18:47). 
Слова Аллаха:
{ إِنَّ جَهَنَّمَ كَانَتْ مِرْصَاداً }
«Воистину, Геенна является засадой», т.е. находится в готовности и ожидании. 
{ لِلطَّـٰغِينَ }
«Для тех, кто преступает границы дозволенного». Это те, кто ослушался, отвергал и противился посланникам. 
{ مَـئَاباً }
«И местом возвращения», т.е. конечным пунктом, пристанищем и вечной обителью. Аль-Хасан и Катада сказали, что никто не попадет в Рай, не преодолев Ад. И если будет у него награда, то он спасется, в противном случае он будет заключен в него, т.е. в Ад. 
{ لَّـٰبِثِينَ فِيهَآ أَحْقَاباً }
«Они пробудут там долгие годы». Толкователи расходятся во мнении, какой это промежуток времени. Ибн Джарир сообщил, что Али ибн Абу Талиб спросил Хиляля аль-Хиджри: «Что значит в Коране "долгие годы"?» Он ответил: «Это восемьдесят лет. В каждом году двенадцать месяцев, в каждом месяце тридцать дней, а каждый год — тысяча лет». Аль-Хасан и Судди считали, что данный период составляет семьдесят лет. 'Абдулла ибн Амру высказал предположение, что это сорок лет, где каждый день равен тысяче лет. Башир ибн Ка'аб сказал по этому поводу: «Мне стало известно, что один долгий год составляет триста лет по двенадцать месяцев в каждом году. В каждом году по триста шестьдесят дней. И каждый год как тысяча лет». Судди прокомментировал: «Каждый долгий год составляет семьдесят лет по триста шестьдесят дней, а каждый год длится словно тысячу лет». Халид ибн Ма'дан сказал, что данный аят и другой аят
{ إِلاَّ مَا شَآءَ رَبُّكَ }
 «если только не пожелает твой Господь (иного)» (11:107) относятся к людям Единобожия, как пишет Ибн Джарир. Также Салем сообщил: «Я слышал, что аль-Хасан, когда его спрашивали о значении аята „Они пробудут там долгие годы", ответил, что фраза "долгие годы" не имеет определенного промежутка времени, это общее понятие вечности в Аду». Катада сказал: «Это бесконечность. Когда заканчивается одна эра, наступает другая». По словам ар-Раби' ибн Анаса, только Аллах знает, какой это промежуток времени. 
Слова Аллаха:
{ لاَّ يَذُوقُونَ فِيهَا بَرْداً وَلاَ شَرَاباً }
«Не вкушая ни прохлады, ни питья», т.е. они не найдут в Преисподней (Аду) ни прохлады для их сердец, ни благого питья, чтобы испить. Поэтому Аллах сказал: 
{ إِلاَّ حَمِيماً وَغَسَّاقاً }
«А только кипяток и гной». Абу аль-Алия сказал: «Исключил из прохлады кипяток, а из питья — гной». Ар-Раби' ибн Анас истолковал: «Что касается кипятка, то это жара, которая достигла своей максимальной отметки и закипела. А гной — это гной с кровью, пот и слезы обитателей Ада. Он невыносимо холодный и зловонный». 
Слова Аллаха:
{ جَزَآءً وِفَـٰقاً }
«Это будет подобающим возмездием». Иными словами, они заслужили такое наказание за свои злодеяния, которые они совершали в земной жизни. 
Слова Аллаха:
{ إِنَّهُمْ كَانُواْ لاَ يَرْجُونَ حِسَاباً }
«Воистину, они не ожидали расчета», т.е. не думали, что наступит Другая жизнь, в которой они поплатятся за свои поступки. 
{ وَكَذَّبُواْ بِـآيَـٰتِنَا كِذَّاباً }
«И полностью отрицали Наши знамения», т.е. отвергали доказательства Аллаха, которые Он ниспослал Своему Посланнику (да благословит его Аллах и приветствует), встретив их с упрямством. 
Слова Аллаха:
{ وَكُلَّ شَىْءٍ أَحْصَيْنَـٰهُ كِتَـٰباً }
«Мы же всякую вещь подсчитали и записали», т.е. Мы знали обо всех делах рабов и все их записывали. И за добро Мы ответим добром, а за зло — злом. 
Слова Аллаха:
{ فَذُوقُواْ فَلَن نَّزِيدَكُمْ إِلاَّ عَذَاباً }
«Вкушайте же! Мы не прибавим вам ничего, кроме мучений». Эти слова будут адресованы обитателям Ада, чтобы они испробовали то, в чем они оказались, а их наказания будут только увеличиваться. Как отметил Катада, это самый суровый коранический аят, ниспосланный Аллахом об обитателях Ада, 
{ فَذُوقُواْ فَلَن نَّزِيدَكُمْ إِلاَّ عَذَاباً }
«Вкушайте же! Мы не прибавим вам ничего, кроме мучений», т.е. муки, которые будут вечно увеличиваться.
{ إِنَّ لِلْمُتَّقِينَ مَفَازاً } * { حَدَآئِقَ وَأَعْنَاباً } * { وَكَوَاعِبَ أَتْرَاباً } * { وَكَأْساً دِهَاقاً } * { لاَّ يَسْمَعُونَ فِيهَا لَغْواً وَلاَ كِذَّاباً } * { جَزَآءً مِّن رَّبِّكَ عَطَآءً حِسَاباً }
31. Воистину, богобоязненных ожидает успех,  32. Райские сады и виноградники,  33. и полногрудые сверстницы,  34. и полные чаши.  35. Они не услышат там ни пустословия, ни лжи.  36. Это будет воздаянием и исчисленным даром от твоего Господа, 
Сказал Всевышний сообщая о том, что Он приготовил для счастливых (обитателей Рая ) из щедрости, милостей и их (вечного) место пребывания: 
{ إِنَّ لِلْمُتَّقِينَ مَفَازاً } 
«Воистину, богобоязненных ожидает успех». Ибн 'Аббас прокомментировал: «Их ожидает место отдыха». Муджахид сказал: «Успех — это спасение от Огня». Очевидно, что ближе к истине толкование ибн 'Аббаса, так как далее Аллах сказал: 
{ حَدَآئِقَ }
«райские сады», т.е. пальмовые и другие рощи, 
{ وَأَعْنَـٰباً وَكَوَاعِبَ أَتْرَاباً }
«и виноградники, и полногрудые сверстницы», т.е. пышногрудые гурии. Ибн Аббас, Муджахид и другие сказали : 
{ وَكَوَاعِبَ }

 «и полногрудые»  Т.е юные девушки, груди которых уже выделяются из-за того, что налились, девственницами равных лет т.е «сверстницы» . Объяснение чего уже приходило в суре " аль-Вакыя "
Слова Аллаха:
{ وَكَأْساً دِهَاقاً }
«И полные чаши». Ибн 'Аббас сказал: «Постоянно наполненные». 'Икрима сказал: «Чистые». Муджахид и аль-Хасан истолковали: «Это значит "наполненные до краев"». 
Слова Аллаха:
{ لاَّ يَسْمَعُونَ فِيهَا لَغْواً وَلاَ كِذَٰباً }
«Они не услышат там ни пустословия, ни лжи». Как еще сказал Аллах: 
{ لاَّ لَغْوٌ فِيهَا وَلاَ تَأْثِيمٌ }
«которое не принесет ни празднословия, ни греха» (52:23). Там не будет праздных и бесполезных разговоров, греха и лжи. Это будет дом мира, в котором не будет никаких недостатков. 
{ جَزَآءً مِّن رَّبِّكَ عَطَآءً حِسَاباً }
«Это будет воздаянием и исчисленным даром от твоего Господа», Аллах воздал им достаточным и полным воздаянием. Так например, говорится: «ахсабту фулянан», что значит «Я дал ему в достаточном количестве, что он сам говорит: «Мне хватит». И от этого же приходят слова: 
(ﺣﺴﺒﻲ ﺍﻟﻠﻪ )

" Хасбия Ллах " (Достаточно мне Аллаха).
{ رَّبِّ ٱلسَّمَٰوَٰتِ وَٱلأَرْضِ وَمَا بَيْنَهُمَا ٱلرَّحْمَـٰنِ لاَ يَمْلِكُونَ مِنْهُ خِطَاباً } * { يَوْمَ يَقُومُ ٱلرُّوحُ وَٱلْمَلاَئِكَةُ صَفّاً لاَّ يَتَكَلَّمُونَ إِلاَّ مَنْ أَذِنَ لَهُ ٱلرَّحْمَـٰنُ وَقَالَ صَوَاباً } * { ذَلِكَ ٱلْيَوْمُ ٱلْحَقُّ فَمَن شَآءَ ٱتَّخَذَ إِلَىٰ رَبِّهِ مَآباً } * { إِنَّآ أَنذَرْنَاكُمْ عَذَاباً قَرِيباً يَوْمَ يَنظُرُ ٱلْمَرْءُ مَا قَدَّمَتْ يَدَاهُ وَيَقُولُ ٱلْكَافِرُ يٰلَيْتَنِي كُنتُ تُرَاباً }
37. Господа небес и земли и того, что между ними, Милостивого, с Которым они не посмеют даже заговорить.  38. В тот день, когда Дух (Джибриль) и ангелы станут рядами, не будет говорить никто, кроме тех, кому позволит Милостивый, и говорить они будут правду.  39. Это будет истинный день, и всякий, кто пожелает, найдет способ вернуться к своему Господу.  40. Мы предостерегли вас от наказания близкого. В тот день человек увидит, что уготовили его руки, а неверующий скажет: "Лучше бы мне быть прахом!" 
Всевышний Аллах говорит о Своем величии, что Он — Господь небес и земли и того, что на них и между ними. Он Милостивый, милость Которого охватывает все сущее. 
{ لاَ يَمْلِكُونَ مِنْهُ خِطَاباً }
«С Которым они не посмеют даже заговорить», т.е. никто не может заговорить с Ним без Его разрешения. Подобно этому Аллах сказал: 
{ مَن ذَا ٱلَّذِى يَشْفَعُ عِندَهُ إِلاَّ بِإِذْنِهِ }
«Кто станет заступаться перед Ним без Его дозволения?» (2:255). И сказал также:
{ يَوْمَ يَأْتِ لاَ تَكَلَّمُ نَفْسٌ إِلاَّ بِإِذْنِهِ }
«Когда этот день наступит, ни один человек не заговорит без Его соизволения»
(11:105)
Слова Аллаха:
{ يَوْمَ يَقُومُ ٱلرُّوحُ وَٱلْمَلَـٰئِكَةُ صَفّاً لاَّ يَتَكَلَّمُونَ }
«В тот день, когда Дух (Джибриль) и ангелы станут рядами, не будет говорить ни-кто». Толкователи Корана расходятся во мнении, кто здесь имеется в виду под Духом. Некоторые утверждают, ссылаясь на Ибн 'Аббаса, что это души потомков Адама. Другие полагают, что это создания, сотворенные по образу и подобию людей, но они не являются ни людьми, ни ангелами. Такое предположение выдвинули ибн 'Аббас и Муджахид. 
Аш-Ша'би, Са'ид ибн Джубайр и ад-Даххак сказали, что Дух — это Джибриль. Как сказал ибн 'Аббас, это величественный архангел. Я (ибн Касир) считаю вариант толко-вания, что это потомки Адама, сомнительным. Очевидно, что здесь имеется в виду Джибриль (мир ему). Эту версию также подтверждает другой аят: 
{ نَزَلَ بِهِ ٱلرُّوحُ ٱلأَمِينُ عَلَىٰ قَلْبِكَ لِتَكُونَ مِنَ ٱلْمُنْذِرِينَ }
«Верный Дух (Джибрил) сошел с ним на твое сердце, чтобы ты стал одним из тех, кто предостерегает» (26:193-194). Мукатиль ибн Хайян сказал : «" ар-Рух" (Дух) это самый почтенный из ангелов и самый приближенный к Господу Всемогущиму и Великому. Также это тот, через кого было дано Откровение» .
Слова Аллаха:
{ إِلاَّ مَنْ أَذِنَ لَهُ ٱلرَّحْمَـٰنُ }
«Кроме тех, кому позволит Милостивый». Подобно этому Аллах сказал: 
{ يَوْمَ يَأْتِ لاَ تَكَلَّمُ نَفْسٌ إِلاَّ بِإِذْنِهِ }
«Когда этот день наступит, ни один человек не заговорит без Его соизволения» (11:105). В «Сахихе» также сообщается: 
" ولا يتكلم يومئذ إلا الرسل " 
«В тот день никто не заговорит, кроме посланников». ( Фатх аль-Бари 13/430 )
{ وَقَالَ صَوَاباً }
«И говорить они будут правду». По мнению 'Икримы, правда заключается в словах «Нет божества, кроме Аллаха». 
Слова Аллаха:
{ ذَلِكَ ٱلْيَوْمُ ٱلْحَقُّ }
 «Это будет истинный день», т.е. он непременно наступит. 
{ فَمَن شَآءَ ٱتَّخَذَ إِلَىٰ رَبِّهِ مَـئَاباً }
«И всякий, кто пожелает, найдет способ вернуться к своему Господу», т.е. найдет дорогу - путь, который приведет его к Нему. 
Слова Аллаха:
{ إِنَّآ أَنذَرْنَـٰكُمْ عَذَاباً قَرِيباً }
«Мы предостерегли вас от наказания близкого», т.е. Дня Воскресения, который уже грядет. 
{ يَوْمَ يَنظُرُ ٱلْمَرْءُ مَا قَدَّمَتْ يَدَاهُ }
«В тот день человек увидит, что уготовили его руки». Человеку будут представлены все его дела, хорошие и плохие, старые и новые. Как еще сказал Аллах: 
{ وَوَجَدُواْ مَا عَمِلُواْ حَاضِرًا }
«Они обнаружат перед собой все, что совершили» (18:49), а также:
{ يُنَبَّأُ ٱلإِنسَـٰنُ يَوْمَئِذِ بِمَا قَدَّمَ وَأَخَّرَ }
«В тот день человеку возвестят о том, что он совершил заранее и что оставил после себя» (75:13). 
{ وَيَقُولُ ٱلْكَافِرُ يَـٰلَيْتَنِى كُنتُ تُرَٰباً }
«А неверующий скажет: "Лучше бы мне быть прахом!"». В тот день, когда неверующий увидит наказание от Аллаха и посмотрит на свои греховные дела, записанные благородными ангелами-посланцами, он пожелает себе смерти, он пожалеет, что был сотворен. Неверующий пожелает себе это, когда он увидит, как Аллах будет судить между животными, жившими на Земле. Он будет судить между ними справедливо и беспристрастно. Даже безрогая овца отомстит за себя овце с рогами. Когда Он закончит над ними (животными) Суд, Он скажет им: «Будьте прахом!». И они станут прахом. Тогда неверующий воскликнет: 
{ يَـٰلَيْتَنِى كُنتُ تُرَٰباً }
«Лучше бы мне быть прахом!». Другими словами: «Лучше бы я был животным и превратился в прах!». На эту тему также существует много широко известных хадисов и преданий от Абу Хурайры, 'Абдуллы ибн 'Амру и др.
Это конец толкования суры «ан-Наба’(Весть)». Вся хвала и благодарность Аллаху - Господу миров!
